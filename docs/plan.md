## 三、技术方案

### 3.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **前端** | Vue 3 + TypeScript + Vite | 组件化架构，类型安全 |
| **后端** | Python + FastAPI | 高性能异步 API |
| **AI 模型** | rembg (U2-Net) | 开源抠图模型 |
| **通信** | REST API | multipart/form-data |

### 3.2 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (Web Browser)                    │
│                                                             │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐  │
│  │   拖拽上传    │ →  │   预览区域    │ →  │   下载按钮    │  │
│  │  Dropzone    │    │  透明背景图   │    │  Save PNG    │  │
│  └──────┬───────┘    └──────────────┘    └──────────────┘  │
│         │                                                      │
│         │  POST /remove-bg (multipart/form-data)              │
└─────────┼────────────────────────────────────────────────────┘
          │
          ↓
┌─────────────────────────────────────────────────────────────┐
│                      后端 (Python)                           │
│                                                             │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐  │
│  │   FastAPI    │ →  │    rembg     │ →  │   返回 PNG    │  │
│  │   接收上传    │    │   AI 抠图    │    │   图片流     │  │
│  └──────────────┘    └──────────────┘    └──────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 API 接口

| 接口 | 方法 | 参数 | 返回 | 说明 |
|------|------|------|------|------|
| `/remove-bg` | POST | `file: 图片文件` | PNG 图片流 | 移除背景 |
| `/health` | GET | - | JSON | 健康检查 |

### 3.4 数据流

```
用户拖拽/选择/粘贴图片
     ↓
前端校验（格式/尺寸/文件大小）
     ├─ 超过建议 (2000px or 2MB) → 弹窗（自动调优/原图/取消）
     └─ 正常 → 继续
     ↓
前端显示原图预览
     ↓
POST /remove-bg 上传到后端
     ↓
后端接收 → PIL 解析 → 校验尺寸 → 缩放至 800px → 转 RGBA → gc 清理
     ↓
rembg AI 推理（u2net → u2netp → silueta 降级 + 重试 ×2 + 90s 超时）
     ↓
Alpha mask 放大还原到原尺寸 → 合成透明 PNG
     ↓
返回 PNG 图片流 + X-Model-Used 响应头
     ↓
前端展示透明结果 → CompareSlider 对比 → 边缘后期工具编辑
     ↓
背景颜色/模板合成（Canvas 端侧）
     ↓
用户下载（PNG/WebP/剪贴板）
```

## 六、环境准备

### 6.1 系统要求

- Python 3.8+
- Node.js 18+（Vue 3 + Vite 前端）
- 内存 4GB+（运行 rembg 模型）

### 6.2 安装步骤

```bash
# 1. 克隆项目
git clone <repo-url>
cd ai-background-remover

# 2. 安装后端依赖
cd backend
pip install -r requirements.txt

# 3. 安装前端依赖
cd ../frontend
npm install

# 4. 启动后端服务
cd ../backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 5. 启动前端开发服务器（新终端）
cd ../frontend
npm run dev
# 浏览器打开 http://localhost:3000
```
