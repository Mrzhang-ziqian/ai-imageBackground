# AI 背景移除工具 - 项目计划

## 一、产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | AI Background Remover |
| 核心功能 | 拖拽上传图片，AI 自动移除背景，支持预览和下载 |
| 目标用户 | 电商卖家、设计师、普通用户 |
| 对标产品 | [remove-bg.io](https://remove-bg.io/zh-CN) |

---

## 二、功能需求

| # | 功能 | 描述 | 优先级 |

| F06 | 错误处理 | 上传失败、处理失败的提示 | P0 |

## 三、技术方案

### 3.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **前端** | Vue 3 + TypeScript + Vite | 组件化架构，类型安全 |
| **后端** | Python + FastAPI | 高性能异步 API |
| **AI 模型** | rembg (U2-Net) | 开源抠图模型 |
| **通信** | REST API | multipart/form-data |

### 3.2 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (Web Browser)                    │
│                                                             │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐  │
│  │   拖拽上传    │ →  │   预览区域    │ →  │   下载按钮    │  │
│  │  Dropzone    │    │  透明背景图   │    │  Save PNG    │  │
│  └──────┬───────┘    └──────────────┘    └──────────────┘  │
│         │                                                      │
│         │  POST /remove-bg (multipart/form-data)              │
└─────────┼────────────────────────────────────────────────────┘
          │
          ↓
┌─────────────────────────────────────────────────────────────┐
│                      后端 (Python)                           │
│                                                             │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐  │
│  │   FastAPI    │ →  │    rembg     │ →  │   返回 PNG    │  │
│  │   接收上传    │    │   AI 抠图    │    │   图片流     │  │
│  └──────────────┘    └──────────────┘    └──────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 API 接口

| 接口 | 方法 | 参数 | 返回 | 说明 |
|------|------|------|------|------|
| `/remove-bg` | POST | `file: 图片文件` | PNG 图片流 | 移除背景 |
| `/health` | GET | - | JSON | 健康检查 |

### 3.4 数据流

```
用户拖拽图片
     ↓
前端读取文件 → 显示原图预览
     ↓
POST 请求上传到后端
     ↓
后端接收图片 → rembg 处理 → 返回 PNG
     ↓
前端展示透明背景结果图
     ↓
用户点击下载 → 保存为 .png 文件
```

## 六、环境准备

### 6.1 系统要求

- Python 3.8+
- Node.js 18+（Vue 3 + Vite 前端）
- 内存 4GB+（运行 rembg 模型）

### 6.2 安装步骤

```bash
# 1. 克隆项目
git clone <repo-url>
cd ai-background-remover

# 2. 安装后端依赖
cd backend
pip install -r requirements.txt

# 3. 安装前端依赖
cd ../frontend
npm install

# 4. 启动后端服务
cd ../backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 5. 启动前端开发服务器（新终端）
cd ../frontend
npm run dev
# 浏览器打开 http://localhost:3000
```

## 九、风险与挑战

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 模型精度不足 | 边缘处理不干净 | 尝试其他模型（U2-Net、 RMBG-1.4） |
| 处理速度慢 | 用户等待时间长 | 添加缓存、优化图片尺寸 |
| 大图处理内存溢出 | 无法处理高分辨率图 | 限制图片大小、增加错误提示 |
